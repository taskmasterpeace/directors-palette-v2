# Storybook

**Route:** `/` (sidebar tab: Storybook)
**Auth Required:** Yes

Create illustrated children's books with AI-generated artwork. Features:

- **Character Builder:** Main character setup with name, age (dropdown), and visual description. The description helps the AI maintain character consistency across all pages
- **Additional Characters:** Add siblings, friends, pets, teachers, parents, or "Other" character types with their own visual descriptions (up to 8 additional characters)
- **Character Avatars:** Visual icons for each character type (child, parent, teacher, pet, etc.)
- **Story Flow:** Multi-step wizard with numbered page indicators at the top
- **Page-by-Page Generation:** Each story page gets its own AI-generated illustration
- **Save/Load/New:** Project management for saving and loading storybook projects
- **Recipes Integration:** Uses pre-built book cover and illustration recipes for consistent style
