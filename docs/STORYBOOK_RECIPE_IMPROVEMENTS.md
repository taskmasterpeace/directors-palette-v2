# Storybook Recipe Prompt Improvements

Copy these improved prompts into **Admin Panel → Recipes Tab → Edit**.

---

## 1. Storybook Style Guide

**Recipe**: `Storybook Style Guide`
**What to edit**: Stage 0 Template

### Improved Template:

```
Create a children's book style guide as a 6-image grid (2 rows × 3 columns) in <<STYLE_NAME:text!>> style.

CRITICAL: Separate each cell with a SOLID BLACK LINE (4-6 pixels wide).

<<STYLE_DESCRIPTION:text>>

CHILDREN'S BOOK STYLE PRINCIPLES:
- Soft, rounded edges on all elements (no sharp corners)
- Warm, inviting color palette with gentle saturation
- Expressive, slightly exaggerated facial features
- Clear silhouettes readable at small sizes
- Whimsical, friendly character proportions
- Rich textures that feel hand-crafted

THE 6 TILES (2x3 grid):
1. CHARACTER CLOSE-UP: A child character headshot, warm golden-hour lighting, 3/4 view, expressive eyes, friendly smile
2. ACTION SCENE: A different character in playful dynamic pose, soft motion blur, joyful energy
3. COZY INTERIOR: Warm interior scene with dappled afternoon light through curtains, inviting atmosphere
4. FRIENDSHIP MOMENT: Two diverse characters (different ethnicities) sharing a kind interaction
5. PLAYFUL POSE: Character mid-movement, whimsical angle, sense of wonder
6. MAGICAL SETTING: Dreamy exterior establishing shot with varied background characters

DIVERSITY REQUIREMENTS:
- Use diverse characters with different ethnicities, skin tones, and appearances
- Show variety in hair colors, styles, and physical features
- Include both male and female characters across the tiles
- Make each character visually distinct and unique

CHILDREN'S BOOK QUALITY:
- Professional picture book illustration standard
- Soft brushwork or clean vector-style edges (per style)
- Colors that pop but aren't harsh on young eyes
- Details that reward re-reading

Output a 16:9 image with exactly 6 tiles (2 rows x 3 columns) separated by black lines.
<<STYLE_NAME:text!>> children's book illustration style throughout all tiles.
```

## 2. Storybook Page (First)

**Recipe**: `Storybook Page (First)`
**What to edit**: Stage 0 Template

### Key additions to add after "CRITICAL STYLE REQUIREMENTS:":

```
CHILDREN'S BOOK ILLUSTRATION TECHNIQUES:
- Soft, rounded linework (no harsh edges)
- Warm color temperature with gentle shadows
- Expressive character faces with large, emotive eyes
- Whimsical proportions (slightly larger heads, expressive hands)
- Textured backgrounds that feel hand-painted
- Gentle vignetting to focus attention
- Leave breathing room for text overlay (typically top or bottom third)
```

## 3. Storybook Page (Continuation)

**Recipe**: `Storybook Page (Continuation)`
**What to edit**: Stage 0 Template

### Key additions:

```
PAGE FLOW CONTINUITY:
- Maintain exact character appearance from previous pages
- Keep consistent lighting direction throughout the book
- Use similar composition balance as earlier pages
- Color palette should feel like the same "world"

CHILDREN'S BOOK PAGE TECHNIQUES:
- Soft, welcoming atmosphere
- Clear focal point on character action
- Background details that support (not distract from) the story
- Age-appropriate expressions and body language
- Warm, nurturing color choices
```

## 4. Storybook Character Sheet

**Recipe**: `Storybook Character Sheet`
**What to edit**: Final stage template

### Add to character sheet generation:

```
CHARACTER SHEET FOR CHILDREN'S BOOKS:
- Friendly, approachable expressions in all poses
- Soft, rounded features (no angular or intimidating elements)
- Clear, readable silhouettes
- Consistent proportions across all expression tiles
- Warm skin tones and inviting color choices
- Clothing with simple, memorable design elements
```

## How to Apply

1. Go to **Admin Panel** (sidebar)
2. Click **Recipes** tab
3. Find the recipe (e.g., "Storybook Style Guide")
4. Click **Edit** (pencil icon)
5. Find the **Stage 0** template
6. Replace or add the improved text
7. Click **Save**

Changes take effect immediately on the next generation.
