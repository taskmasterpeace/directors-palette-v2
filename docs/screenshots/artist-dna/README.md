# Artist Lab (Artist DNA)

**Route:** `/music-lab/artist-dna`
**Auth Required:** Yes

Artist profile and visual identity management for music projects. Features:

- **Artist Cards:** Grid of artist profiles, each showing:
  - Artist name, genre, and avatar
  - Character sheet thumbnails (turnaround views, expressions)
  - "Make Up" button for creating character sheets
  - Edit and settings controls
- **Create New Artist:** Button to add a new artist with name, genre, description, and reference photos
- **Start from Reel Artist:** Quick-start option to base a new artist on an existing one
- **Character Sheet Integration:** Each artist can have multiple character reference sheets for consistent visual generation across music video shots
