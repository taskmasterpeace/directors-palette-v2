# Shot Animator

**Route:** `/` (sidebar tab: Shot Animator)
**Auth Required:** Yes

Converts still images into short animated videos. Features:

- **Model Selector:** Choose video generation models (Seedance 1.5 Pro, Kling, Wan, etc.) with cost badges
- **Image Upload Zone:** Click-to-upload or "Browse Gallery" to select a source image from existing generations
- **Settings:** Duration, resolution, upscale options, and a search filter for reference images
- **Upload Button:** Direct file upload support
- **Video Gallery:** Right panel shows previously generated videos with play controls
- **Video Limit:** Progress indicator showing usage against the 100-video user limit
