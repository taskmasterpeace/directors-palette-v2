# Canvas Editor

**Route:** `/` (sidebar tab: Canvas Editor)
**Auth Required:** Yes

A freeform canvas for compositing and editing generated images. Features:

- **Canvas Area:** Large white canvas workspace with adjustable background color
- **Drawing Tools:** Select, Mask, Rectangle, Arrow, Text, Eraser tools in the bottom toolbar
- **Canvas Settings Panel:** Background color picker, canvas dimensions (default 1024x1024)
- **Actions Panel:** "Place Image" button to import generated images onto the canvas
- **Export & Share:** Export the final composition in various formats
