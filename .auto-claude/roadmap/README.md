# Roadmap Directory

This directory contains project roadmap and planning documents.

## Structure

```
roadmap/
├── current.md           # Current development focus
├── backlog.md           # Future features backlog
└── completed/           # Archived completed features
    ├── q4-2024.md
    └── ...
```

## Roadmap Format

### Feature Entry
```markdown
## Feature Name

**Priority**: High/Medium/Low
**Complexity**: Simple/Moderate/Complex
**Dependencies**: [list of dependent features]

### Description
Brief description of the feature.

### Acceptance Criteria
- [ ] Criterion 1
- [ ] Criterion 2

### Technical Considerations
Notes on implementation approach.
```

## Current Focus Areas

### Core Generation
- Image generation with multiple models
- Video generation from reference images
- Recipe system for automated workflows

### User Experience
- Gallery management and organization
- Prompt library and wildcards
- Reference image management

### Platform
- Credit system and billing
- Admin dashboard
- Community features

## Adding to Roadmap

1. Create a feature entry in `backlog.md`
2. Prioritize and move to `current.md` when ready
3. Archive to `completed/` when done
