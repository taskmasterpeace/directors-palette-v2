# Brand Studio

**Route:** `/` (sidebar tab: Brand Studio)
**Auth Required:** Yes

AI-powered brand identity creation and management tool. Features:

- **Brand Selector:** Dropdown to switch between multiple brands, plus "New Brand" creation
- **Brand Score:** Letter grade (A+, A, B, etc.) rating the completeness of the brand identity
- **Tabs:** Visual Identity, Voice & Messaging, Assets, Brand Fonts
- **Visual Identity Panel:**
  - Color Palette: Primary, secondary, accent, and neutral colors with hex codes
  - Typography: Font family selection (e.g., Bangers, Open Sans) with size/weight previews
  - Visual Style: Overall aesthetic direction for the brand
- **Logo Display:** Shows the brand logo with the brand name
- **Online/Tagline:** Brand tagline and online presence links
- **Create Section:** Generate new brand assets using AI
