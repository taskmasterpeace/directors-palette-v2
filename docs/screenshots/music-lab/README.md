# Music Lab

**Route:** `/music-lab`
**Auth Required:** Yes

Music video treatment creation tool. Scout tracks, commission AI directors, and greenlight your favorite vision. Features:

- **Artist Selector:** Choose from existing artists (e.g., Sasha Foxworth, Vexor, Khadijah Tomorrow, Country Fat) or create a new one
- **Studio Dashboard:** Central hub for the music video creation workflow
- **Step 1 — The Track:** Audio file upload zone (MP3, WAV, M4A, AAC up to 50MB)
- **Step 2 — The Lyrics:** Paste or import lyrics with AI section detection
- **Step 3 — Writing Studio:** Song structure editor with verse/chorus/bridge sections
- **Step 4 — AI Directors:** Commission multiple AI directors to create competing visual treatments
- **Step 5 — Greenlight:** Compare director treatments and approve the final vision
