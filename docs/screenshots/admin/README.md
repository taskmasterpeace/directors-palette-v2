# Admin Dashboard

**Route:** `/admin`
**Auth Required:** Yes (admin only)

Administrative panel for managing users, analytics, and platform settings. Features:

- **Tab Navigation:** Users, Activity, Coupons, Storybooks, Community, Splash, Resources
- **Stats Cards:** Total Users, Pts Purchased, Pts Used, Total Revenue — each with an icon and dollar/count value
- **User Table:** Searchable, sortable table of all users with columns:
  - Email, Balance, Score, Purchased, Used, Role
  - Role badges (Grant, Admin, User) color-coded
  - Search by email and status filter
- **User Management:** Grant pts, change roles, view user details
- **Revenue Tracking:** Real-time revenue and usage analytics
