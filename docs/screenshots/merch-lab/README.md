# Merch Lab

**Route:** `/` (sidebar tab: Merch Lab)
**Auth Required:** Yes

Design and preview custom merchandise using generated artwork. Features:

- **Product Type Selector:** Choose product templates (t-shirt shown by default, with additional product types)
- **T-Shirt Preview:** Live mockup showing front/back with placement zones for graphics
- **Graphic Placement:** "Center Graphic" and "Left Chest" placement options
- **Color Picker:** Full color grid for selecting garment color
- **Design Area:** "Your design appears here" zone for uploading/placing artwork
- **Printify Integration:** "Make on Printify" button to send designs to print-on-demand production
- **Cost Display:** Shows pts cost for design generation
