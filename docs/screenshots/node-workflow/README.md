# Node Workflow

**Route:** `/` (sidebar tab: Node Workflow) or `/node-workflow`
**Auth Required:** Yes

A visual node-based editor for building complex generation pipelines. Drag-and-drop nodes onto a canvas and connect them to create multi-step workflows. Node types:

- **Input:** Upload reference images
- **Prompt:** Text prompt with optional variables
- **Generation:** Generate images with AI models
- **Tool:** Apply image processing (resize, filter, composite)
- **Output:** Final result output

The canvas shows node count, connection count, and provides quick tips for usage. Nodes connect via handles to define data flow.
