# Community

**Route:** `/` (sidebar tab: Community)
**Auth Required:** Yes

Browse, share, and download community-created content. Features:

- **Content Type Tabs:** All, Wildcards, Recipes, Prompts, LoRAs, Directors
- **Search & Filter:** Search bar with category and sort (Most Popular, Newest, etc.) dropdowns
- **Content Cards:** Grid of community items showing:
  - Thumbnail preview
  - Name and "Official" badge for system content
  - Type badge (Recipe, LoRA, etc.)
  - Creator info
  - Stage/field count
  - Star ratings
  - Style tags and trigger words (for LoRAs)
  - Green checkmark if already added to your library
- **Create Recipe Button:** Submit your own recipes to the community
- **Refresh Button:** Reload community listings
- **Add to Library:** One-click to add community items to your personal collection
