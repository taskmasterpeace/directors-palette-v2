## Features Deployed Today (Nov 20)

### New Tabs Added
1. **Gallery Tab** - Standalone image gallery with:
   - 48 images per page
   - Select All button
   - Full viewport height
   - Search, delete, bulk operations

2. **Story Creator Tab** - Story-to-shots workflow with:
   - Story Input section (paste story, extract shots)
   - Shots Review section (edit prompts, add @references)
   - Generation Queue section (batch generate, progress tracking)

### @Reference Autocomplete (Shot Creator)
- Type `@` to trigger autocomplete
- Shows tagged images (@hero, @villain) and categories (@people, @places, @props, @layouts)
- Keyboard navigation (arrows, Enter, Escape)
- Mobile-responsive
