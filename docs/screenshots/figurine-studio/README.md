# Figurine Studio

**Route:** `/` (sidebar tab: Figurine Studio)
**Auth Required:** Yes

Converts 2D character images into 3D models using Tencent Hunyuan 3D. Features:

- **Source Image Upload:** Drop or browse for a character image. Can also paste image URLs directly
- **3D Preview Panel:** Interactive 3D model viewer (powered by @google/model-viewer) with zoom, pan, rotate controls and background color presets
- **Generate Button:** "Generate 3D Model" with pts cost display (25 pts)
- **Gallery Integration:** "Browse Gallery" to select from existing generated images
