# Gallery

**Route:** `/gallery`
**Auth Required:** Yes

Unified gallery for browsing all generated images and videos across every feature. Features:

- **Source Filters:** All, Shot Creator, Storybook, Storyboard, Artist DNA tabs
- **Search:** Full-text prompt search across all generations
- **View Modes:** Grid (small), Grid (large), and Single-image view toggles
- **Image Counter:** Shows current count vs. 500-image storage limit
- **Pts Balance:** Displays remaining pts in the header
- **Select Mode:** Multi-select for batch operations (delete, download, share)
- **Empty State:** "No images generated yet" prompt linking to Shot Creator
- **Fullscreen Lightbox:** Click any image to view full-size with metadata, prompt, and action buttons (Make Figurine, Animate, Share, Delete)
